# 📌 About

This repository is a structured collection of my daily engineering work focused on modern infrastructure.

Instead of simply completing courses or collecting certificates, I learn by building, breaking, troubleshooting, and documenting real systems.

The repository is organized into three parallel engineering tracks:

* ☁️ **Cloud Engineering**
* 🔒 **Security Engineering**
* 📈 **Site Reliability Engineering (SRE)**

Every topic is explored through hands-on labs, terminal sessions, code, and technical documentation.

# 💡 Engineering Principles

This repository follows a simple engineering philosophy:

* Build before reading.
* Learn from failures.
* Debug using first principles.
* Document everything.
* Improve through consistent daily practice.

The goal is not to memorize commands, but to understand how infrastructure behaves under real-world conditions.
